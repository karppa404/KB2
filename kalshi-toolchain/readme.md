# Kalshi-Toolchain
The kalshi tool chain is a command line utility to trade on the Kalshi network from the terminal. The main use case for this is to enable LLM's to interact with Kalshi. The functional use of this tool chain can be placed into three buckets:
- Buy / Sell Tickers
  - Models with one command can liquidate a position or take a new position
- Inspect portfolio
  - models can look at outstanding positions 
  - models can see past positions
  - models can see the preformance of a portfolio overtime
- Search for markets availible & the web for information
  - models can search kalshi for open markets
